# Screenshots

README では次の画像パスを参照します。

- `screenshots/home.png`
- `screenshots/vehicles.png`
- `screenshots/drivers.png`
- `screenshots/operation-tasks.png`
- `screenshots/dashboard.png`

実行環境で Next.js を起動した後、以下で画像を生成できます。

```bash
npm --prefix frontend install
npm install
npx playwright install chromium
npm run screenshots
```

この Portfolio では実在サービス名や実データを使わず、中立的な mock/seed data の画面を撮影します。
